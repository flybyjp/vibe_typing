# タイピング対戦ゲーム 実装プラン

## 概要
Weather Typing を参考にした、Webブラウザベースの2人対戦タイピングゲーム

## 確定仕様

### 基本仕様
- **プラットフォーム**: Webブラウザ
- **対戦形式**: 2人対戦、文章完了型（先に打ち終わった方が勝ち）
- **マッチング**: ルーム制（ホストがルーム作成、相手がルームIDで参加）
- **ユーザー管理**: 不要（ゲスト対戦のみ、名前入力のみ）

### ゲームルール
- **ラウンド数**: ホストが設定可能（1, 3, 5本勝負など）
- **勝敗判定**: 各ラウンドで先に文章を打ち終わった方がポイント獲得
- **入力方式**: ローマ字のみ
- **ローマ字変換**: 複数パターン許容（si/shi、tu/tsu、ltu/xtu/子音連打など）

### ネットワーク
- **接続方式**: サーバーアドレス指定（URL/IP入力で切り替え）
- **インターネット対戦**: 公開サーバーに接続
- **LAN対戦**: ローカルサーバーのアドレスを指定

### ルーム設定（ホストが設定可能）
- ラウンド数（1, 3, 5, 7, 9 など）
- 問題セット（複数のCSVから選択）

### 画面・UI
- **対戦画面**:
  - 出題文（日本語）と読みがな表示
  - 入力済み/未入力の文字を色分け表示
  - 自分と相手のプログレスバー
- **結果画面**:
  - 勝敗表示
  - タイプ速度（KPM: Keys Per Minute）
  - 正確率
  - クリアタイム

### 音声
- 効果音のみ（タイプ音、正解/ミス音、勝敗音）

### 出題データ
- **CSV形式**: `日本語文,読みがな`
- 例: `今日は良い天気です,きょうはよいてんきです`
- 複数の問題セット（CSV）を登録可能

## 技術スタック

### フロントエンド
- **React** (TypeScript)
- **Socket.io-client** (リアルタイム通信)
- **CSS Modules**

### バックエンド
- **Node.js** (TypeScript)
- **Socket.io** (WebSocket通信)
- **Express** (静的ファイル配信、API)

## 起動方法

```bash
# 依存関係のインストール（初回のみ）
npm run install:all

# 開発サーバー起動
npm run dev
```

- サーバー: http://localhost:3001
- クライアント: http://localhost:5173
- LAN内の他端末からは `http://<ホストのIPアドレス>:5173` でアクセス

## 検証方法

### ローカル検証
1. サーバー起動（`npm run dev`）
2. ブラウザを2つ開いてアクセス
3. 一方でルーム作成、もう一方でルーム参加
4. 対戦が正常に動作することを確認
5. 統計データが正しく表示されることを確認

### LAN対戦検証
1. サーバー起動（`npm run dev`）
2. 同一LAN内の別端末から `http://<ホストIP>:5173` にアクセス
3. 別端末側でサーバー設定を `http://<ホストIP>:3001` に変更
4. ルーム作成・参加して対戦できることを確認
